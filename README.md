# n8n AI Daily Tip Automation

This project uses n8n to send a daily AI-generated productivity tip via email.

## Workflow

Schedule Trigger → OpenAI → Send Email

## Features

* Daily automated trigger
* AI-generated productivity tips
* Email delivery
* JSON parsing in n8n
* No-code workflow automation

## Tech Used

* n8n (cloud)
* OpenAI API
* SMTP Email

## Example Output

"Prioritize your tasks: focus on the top three things that will make the biggest impact today."
